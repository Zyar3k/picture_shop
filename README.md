# Picture shop

#### [LIVE DEMO](https://picture-shop.netlify.app/)

> Website with elements of the photographer's portfolio. Functioning shopping cart and validated message form
##### Used in project
- autoprefixer-cli
- browser-sync
- globstar
- mkdirp
- node-sass
- npm-run-all
- nu-html-checker
- onchange
- javascript ES5


### Installation


To run app
```sh
cd picture_shop
npm run init-project
npm run watch
```
